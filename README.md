# Professor X Downloader 📥

YouTube + Instagram video downloader Android app — powered by yt-dlp.

## Features
- ⬇️ YouTube (Videos, Shorts, Playlists)
- 📸 Instagram (Posts, Reels, Stories)  
- 📋 Download History
- 🌙 Dark / Light Mode
- 🚫 Koi browser nahi khuega
- 🚫 Koi redirect nahi

---

## GitHub Se APK Kaise Banayein (Bina PC ke!)

### Step 1 — GitHub Account
github.com pe jaao aur free account banao

### Step 2 — New Repository
- "New" button dabao
- Name: `professor-x-downloader`
- Public select karo
- "Create repository" dabao

### Step 3 — Files Upload
- "uploading an existing file" link dabao
- Is ZIP ke saare files drag & drop karo
- "Commit changes" dabao

### Step 4 — APK Build Hogi Automatically!
- "Actions" tab pe jaao
- "Build Professor X APK" workflow dikhega
- Green checkmark aane ka wait karo (~5-10 min)

### Step 5 — APK Download
- Workflow pe click karo
- "Artifacts" section mein `Professor-X-Downloader` dikhega
- Download karo → Phone pe install karo!
